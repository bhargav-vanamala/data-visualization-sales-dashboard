# Sales Data Visualization Dashboard

## Objective
Analyze and visualize sales performance to uncover trends, revenue drivers, and business insights using clear and impactful charts.

## What This Project Covers
- Data cleaning and preparation
- Exploratory Data Analysis (EDA)
- Monthly and yearly sales trends
- Region-wise and category-wise performance
- Revenue and profit comparison
- Business-focused visual storytelling

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Outcome
Delivered clear visual insights that help stakeholders understand sales performance and support data-driven decision-making.
